如何开始Hexo

最初只是想要一个个人知识管理库而已，记录和查找起来方便一点。这篇就当一个记录，从一个什么都不会的小白开始，怎样发布第一篇自己的博客。

安装

大致介绍可以参看官网Hexo。这里主要是详细步骤，也参考了很多的其他文章。

安装Git Bash

我的系统是Windows 64位，官网下载速度特别慢，所以推荐地址为Git Bash。双击下载好的exe文件，一路next就好啦~

安装NodeJs

NodeJs,双击打开安装，也是一路next，不过在Custom Setup这一步记得选 Add to PATH ,这样就不用自己去配置电脑上环境变量了。

安装Hexo

创建一个用来存放所有blog的文件夹，双击打开文件夹后，在空白处右键，点击Git Bash here。
安装Hexo命令：$ npm install -g hexo-cli
初始化命令：$ hexo init ,所在的文件夹会出现一些文件，node_module，public，scaffolds，source，themes，config.yml，db.json，package.json

与Github关联配置

在github里创建一个repo，名称为yourname.github.io, 其中yourname是你的github名称，按照这个规则创建才有用。
在gitbash里输入：$ git config –global user.name “YourName”
$ git config –global user.email “YourEmail”
$ ssh-keygen -t rsa -C “YourEmail@example.com“
后续操作直接回车即可
把获取的ssh key 加到 github的settings中的SSH Keys
在gitbash里输入：$ ssh -T git@github.com 验证是否成功

用编辑器（我用的是VSCode)修改_config.yml文件
deploy:
type: git
repo: https://github.com/YourgithubName/YourgithubName.github.io.git
branch: master

安装自动保存上传到github插件
在gitbash里输入：$ npm install hexo-deployer-git –save

在gitbash中输入 hexo clean
hexo generate
hexo server

然后打开浏览器（我用的是Google chrome）输入：http://localhpst:400

新建文章

在gitbash里输入： hexo new ‘文章名’
新建的文章在source/_posts路径下

以上大部分内容参考的文章是https://www.cnblogs.com/visugar/p/6821777.html
